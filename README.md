# Учебный проект

Этот проект был выполнен в учебных целях для практики верстки и организации структуры фронтенд-проекта.

## 🚀 Технологии

- HTML5
- CSS3
- SCSS (Sass)
- JavaScript
- Node.js

## 📂 О проекте

В ходе разработки были закреплены навыки:

- семантической HTML-разметки;
- адаптивной верстки;
- использования SCSS (переменные, миксины, вложенность, организация файлов);
- работы с JavaScript для реализации интерактивности;
- использования Node.js для управления зависимостями и запуска проекта.

## 🛠 Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone <repository-url>
```

2. Перейдите в папку проекта:

```bash
cd <project-name>
```

3. Установите зависимости:

```bash
npm install
```

4. Запустите проект:

```bash
npm start
```

> Если в проекте используется другой скрипт запуска (например, `npm run dev`), замените команду на соответствующую.
